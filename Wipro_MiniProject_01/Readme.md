# Wipro Mini Project 01

A simple Java employee salary calculator that reads an employee number from user input and prints employee details along with their calculated salary.

## Project details

- `wipro_mini_01.java` stores data for employee IDs, names, designations, departments, basic pay, HRA, and income tax.
- The app searches for the entered employee number and prints:
  - Employee number
  - Employee name
  - Department
  - Designation
  - Calculated salary
- Salary is computed using a base amount plus HRA and a designation-specific allowance, minus income tax.

## How to compile

Open a terminal in this project folder and run:

```bash
javac wipro_mini_01.java
```

## How to run

After compiling, run:

```bash
java wipro_mini_01
```

Then enter an employee number when prompted.
